CSS Variables Customized

Color Variables:
  • --bs-primary: #20c997; (Teal)
  • --bs-secondary: #ff6b6b; (Coral)
  • --bs-success: #9c27b0; (Purple)
  • --bs-info: #2196f3; (Deep Blue)
  • --bs-warning: #ff9800; (Amber)
  • --bs-danger: #ff5722; (Deep Orange)
Other Variables:
  • --bs-border-radius: 0.75rem;
  • --bs-btn-border-radius: 0.75rem;
  • --bs-btn-padding-x: 1.5rem;
  • --bs-btn-font-weight: 600;
  • --bs-box-shadow: 0 0.5rem 1rem rgba(0,0,0,0.1);
  • --bs-font-sans-serif: 'Segoe UI', system-ui;

Note: All Bootstrap components automatically use these custom variables. No additional classes or inline styles needed.
